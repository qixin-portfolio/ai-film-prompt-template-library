# AI影视制作｜提示词模板库 V1（中文版）

一套面向 AI 影视制作的中文提示词生产模板，覆盖首帧生图、图生视频、人物资产、复杂动作、音乐视频、精准嘴型、群像、连续镜头、验收与返修等常见任务。

> 这不是“万能提示词合集”，而是一套按任务类型调用的生产模板。先判断任务属于哪一类，再填入资产、空间、动作、镜头和验收要求。

## 使用原则

- 首帧生图提示词与图生视频提示词分开写。
- 每个镜头只承担一个主要叙事任务。
- 参考资产必须明确“继承什么、不继承什么、使用范围”。
- 状态变化必须持续，终止帧必须明确。
- 生成失败后先诊断问题来源，每轮只修改一个主要变量。
- 人物身份、服装、场景结构、空间关系和连续性优先于质感增强。

## 资产命名示例

```text

通过版本：03-03_APPROVED
```
